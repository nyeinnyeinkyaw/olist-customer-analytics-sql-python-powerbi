# Data Folder

Download the Brazilian E-Commerce Public Dataset by Olist from Kaggle.

Place the CSV files here:

```text
data/raw/
```

Expected file names:

```text
olist_customers_dataset.csv
olist_geolocation_dataset.csv
olist_order_items_dataset.csv
olist_order_payments_dataset.csv
olist_order_reviews_dataset.csv
olist_orders_dataset.csv
olist_products_dataset.csv
olist_sellers_dataset.csv
product_category_name_translation.csv
```

Do not upload large raw data files to GitHub if they make the repository too heavy.
Instead, explain the data source in the main README.
