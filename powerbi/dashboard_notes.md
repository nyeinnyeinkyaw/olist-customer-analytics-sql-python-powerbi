# Power BI Dashboard Notes

Save your Power BI file here after you build the dashboard:

```text
powerbi/olist_customer_analytics_dashboard.pbix
```

Also export screenshots and save them in:

```text
images/
```

Recommended screenshot names:

```text
dashboard_page_1_executive_overview.png
dashboard_page_2_customer_crm_analysis.png
dashboard_page_3_delivery_satisfaction.png
dashboard_page_4_product_performance.png
```

For GitHub and Notion, screenshots are more important than the PBIX file because recruiters can view them quickly.
